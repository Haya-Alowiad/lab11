In this lab, K-Means clustering was used to analyze customer credit card behavior. The data was cleaned by removing the `CUST_ID` column and handling missing values. Then, the features were scaled using `StandardScaler`.

The elbow method and silhouette score were used to choose the best number of clusters, and K = 4 was selected. The model grouped customers into different segments based on spending, balances, and cash advance usage. Finally, PCA was used to visualize the clusters and better understand customer behavior.
